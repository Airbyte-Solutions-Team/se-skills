## What the app is
Local FastAPI + vanilla-JS UI (no build step) over the SE skills suite. `cd webapp && uv run app.py` ‚Üí http://127.0.0.1:8787 (needs `CPATH/LIBRARY_PATH` for portaudio on this Mac ‚Äî see "Run" below). Browse team ‚Üí member's accounts ‚Üí an account's opportunities ‚Üí generated outputs; invoke skills; ask follow-ups on outputs; Live Transcribe a Zoom call with an AI copilot.

```bash
cd ~/airbyte-work/02-repos/se-skills/webapp
```
